# RAG Chatbot using Endee Vector Database

## Project Overview
This project implements a simple Retrieval Augmented Generation (RAG) based chatbot. The system converts text documents into vector embeddings and performs similarity search using vector database concepts inspired by Endee.

## Problem Statement
Traditional keyword search does not capture semantic meaning. This project solves the problem by using vector embeddings to retrieve semantically relevant information.

## System Design
1. Input text documents
2. Convert documents into embeddings
3. Store embeddings in vector format
4. Perform similarity search
5. Retrieve relevant results

## How Endee is Used
Endee is a high-performance vector database. In this project, embeddings generated from Sentence Transformers simulate vector storage and retrieval similar to Endee's vector search mechanism.

## Tech Stack
- Python
- Sentence Transformers
- NumPy
- Endee Vector Database
